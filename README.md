# Emulation of Quantum State Tomography and the Bell Test using Classically Entangled Light Beams

## Abstract

This experiment emulates quantum entanglement, quantum state tomography, and the CHSH Bell test using classical optical equipment -- a pulsed laser with randomised polarisations, waveplates, beam splitters, and webcams (as detectors). The repository contains Python scripts for processing raw video feeds, reconstructing quantum density matrices, simulating tomography measurements, evaluating Bell inequalities, and generating paper figures. Executing the codebase converts recorded video frames into discrete pulse detection counts, reconstructs $4 \times 4$ state density matrices, calculates the CHSH Bell parameter (with S = 2.60), and outputs the signal intensity plots and 3D density matrix bar charts corresponding to Figures 3,6,7 of the paper.

---

## File Descriptions

### `src/p1.py` & `src/p3.py`

* **Description:** Video processing scripts that analyse recorded webcam feeds frame-by-frame. They define regions of interest (ROI) for each detector path, integrate pixel intensities for incoming laser pulses, and apply amplitude thresholding to isolate valid detection events from background noise.
* **Inputs:** Raw video files (.mp4) recorded by the webcam detectors during polarisation measurements.
* **Outputs:** Processed time-series pulse intensity arrays.

---

### `src/tomography.py`

* **Description:** Performs quantum state tomography by reconstructing 4x4 density matrices from detection probabilities.
* **Inputs:** [Intensity vs time] csv files generated by `p1.py`.
* **Outputs:** 4x4 numerical density matrix arrays for the emulated Bell states.

---

### `src/sim.py`

* **Description:** Simulates ideal and noise-perturbed quantum state tomography measurements and density matrix reconstructions for comparison with experimental data.
* **Inputs:** Self-contained.
* **Outputs:** Simulated $4 \times 4$ density matrices and detection probability distributions; optionally, **Figure 4** of the paper -- idelity as a function of the number of simulated bits.

---

### `src/bell-test.py`

* **Description:** Evaluates the CHSH Bell test by calculating measurement correlation functions across 16 polariser angle combinations and computing the overall Bell parameter S.
* **Inputs:** [Intensity vs time] csv files generated by `p3.py`.
* **Outputs:** Correlation values $E(\alpha, \beta)$ and the final Bell parameter S).

---

### `src/graph-feeds.py`

* **Description:** Plots time-series signal intensity versus video frame counts for all polariser angle pairs with both axes normalised.
* **Inputs:** Raw signal intensity arrays from `p1.py` and `p3.py`.
* **Outputs:** **Figure 5** of the paper (a monstrous plot of all 16 graphs).

---

### `src/graph-matrix.py`

* **Description:** Generates 3D column graphs representing the real components of reconstructed $4 \times 4$ density matrices across sequence lengths of 10, 25, and 50 bits.
* **Inputs:** Reconstructed numerical density matrices output by `tomography.py` or `sim.py`.
* **Outputs:** **Figures 3,6,7** of the paper (3D bar charts of the reconstructed density matrices).
